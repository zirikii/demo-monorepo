# Privacy

This is an **unofficial demo** and is not affiliated with Singtel Optus Pty Limited. No
real personal information is collected, stored, or transmitted to any third party.

## What this demo stores

- A mock session cookie signed with a demo secret so protected pages can be viewed.
- Local JSON files used to render plans, bills, recharges, and account settings.

All data resets to the seeded values on a fresh checkout. Any details entered into demo
forms stay on the local machine running the app.

## About this notice

This is a **demonstration application** and is not affiliated with SEEK Limited. It does not collect, transmit or sell any personal information to third parties.

## What this demo stores

All data you enter (saved jobs, notes, applications, profile details and settings) is stored **locally** in JSON files on the machine running the demo. Nothing is sent to an external server.

## Mock authentication

Sign-in is simulated. Any email and password combination is accepted in demo mode, and a signed cookie is used purely to gate the candidate area. Do not enter real passwords.

## Your control

Because all data is local, you can reset the demo at any time by restoring the seed data files in the `data/` directory.
